# smart-room-controller

## Overview
I am creating a smart room contoller that uses hue lights and wemo switches. It features:
1. Automatic modes
2. Manual mode
## Details

This project utilizes:

* Teensy 3.3 microcontroller
* Breadboard
* Wemo switches
* Phillips Hue lights
